# 济钢水厂建设期质量安全 AI 辅助监督与可视化管理系统 Demo

这是一个基于 **React + Vite + TypeScript** 的前端流程演示 Demo，用于展示“济钢水厂建设期质量安全 AI 辅助监督与可视化管理系统”的页面结构、业务闭环、交互逻辑与大屏视觉效果。

> 说明：本项目不包含真实后端、数据库、图片上传服务或 AI 模型；所有业务数据、AI 判断、聊天内容、问题单与整改闭环信息均由前端 mock 数据模拟。

## 项目定位

本 Demo 面向政企项目管理系统演示场景，重点覆盖：

- 首页深蓝科技风驾驶舱；
- 白底蓝色政务风的巡检、问题单、整改流程页面；
- 左侧固定导航 + 顶部项目态势栏；
- 今日任务从巡检范围选择到归档销项的完整闭环；
- AI 智慧助理、隐蔽节点、整改闭环列表等核心业务模块；
- 1920×1080 大屏优先的展示布局。

## 启动方式

### 1. 安装依赖

```bash
npm install
```

### 2. 本地启动

```bash
npm run dev
```

启动后访问终端中显示的本地地址，通常为：

```text
http://localhost:5173
```

### 3. 构建检查

```bash
npm run build
```

## Demo 流程

默认进入“首页总览”。你可以按以下路径体验完整业务闭环：

1. 点击左侧“今日任务”；
2. 在“今日巡检任务”点击“下一步：执行巡检”；
3. 在“执行巡检”页面可点击待上传照片，将状态从“待上传”变为“已上传”；
4. 点击“生成辅助判断”；
5. 在“AI辅助判断”页面点击“进入人工确认”；
6. 在“人工确认”页面点击“生成问题单”；
7. 在“生成问题单”页面点击“提交问题单”；
8. 在“整改跟踪”页面点击“提交复查申请”；
9. 在“整改复查”页面点击“通过销项”；
10. 页面显示“问题已完成闭环并归档。”提示。

- AI 判断依据、整改建议、问题单信息；
- 隐蔽节点台账与 AI 提醒；
- 整改闭环问题列表。

## 后续正式开发建议

若进入真实项目开发，可按以下方向扩展：

1. **真实后端接口**：接入任务、问题单、整改记录、隐蔽节点、消息通知等 REST/GraphQL API；
2. **数据库与权限**：建设项目、标段、角色、班组、点位、问题单、流程实例等核心数据模型；
3. **图片上传与存储**：接入对象存储、图片压缩、EXIF 信息、拍摄点位与水印；
4. **AI 识别服务**：对接安全帽、临边防护、积水、电缆拖地、隐蔽节点证据完整性等识别模型；
5. **流程引擎**：引入任务派发、签收、整改、复查、销项、逾期升级等可配置工作流；
6. **报表与归档**：生成质安日报、隐蔽验收台账、整改闭环台账、审计留痕报告；
7. **多端协同**：扩展移动端巡检、现场拍照、AR 辅助、短信/微信/企业 IM 通知。
